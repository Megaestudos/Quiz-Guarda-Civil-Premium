# PlenAula

Plataforma gratuita de simulados, jornada de estudos, resumos e preparacao para concursos.

## Visao Geral

- App web estatico publicado pelo Firebase Hosting.
- Autenticacao e dados pelo Firebase Auth/Firestore.
- Cloud Functions em Node.js para recursos de IA com Gemini.
- Painel administrativo para alunos, conteudos, simulados e jornada.

## Observacao Sobre Acesso

O projeto e gratuito. Alguns nomes internos legados ainda usam `premium` para CSS, funcoes ou campos antigos, mas eles nao representam cobranca ou assinatura ativa.
